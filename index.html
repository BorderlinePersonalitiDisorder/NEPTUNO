¿Por qué el mundo necesita esto?

El Problema

  • El 98% de los sitios web tienen errores de accesibilidad
  • Componentes complejos requieren soluciones manuales
  • Desarrolladores reinventan la rueda constantemente
  • La experiencia de usuario es inconsistente
  • 1.3 mil millones de personas excluidas digitalmente

Nuestra Solución

  • Elemento HTML nativo para accesibilidad
  • Componentes accesibles desde el primer día
  • Comportamiento consistente en todos los navegadores
  • Reducción drástica del código necesario
  • Inclusión digital para todos

Componentes Accesibles

Carrusel

Navegación por teclado, controles ARIA, pausa automática al enfocar, compatibilidad con lectores de pantalla.

Acordeón

Expansión/colapso accesible, manejo de teclado, semántica adecuada para lectores de pantalla.

Pestañas (Tabs)

Navegación por teclado, indicadores de foco, relación ARIA entre controles y paneles.

Modal

Trampa de foco, cierre con ESC, restauración de foco, fondo no desplazable.
